📝 Classes de Paginação

  • .pagination - Container principal
  • .page-item - Item de página
  • .page-link - Link da página
  • .active - Página atual
  • .disabled - Item desabilitado
  • .pagination-lg / .pagination-sm - Tamanhos
  • .justify-content-center / .justify-content-end - Alinhamento
💡 Dicas de Uso
  • Sempre mostre a página atual com .active
  • Desabilite botões quando não houver próxima/anterior
  • Exiba informações de total de registros
  • Para muitas páginas, use "..." para indicar páginas ocultas
  • Combine com seletor de itens por página para melhor UX

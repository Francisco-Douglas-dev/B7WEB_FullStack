📝 Resumo

  • .col-[número] define a largura em uma escala de 12
  • .col sem número se auto-ajusta ao espaço disponível
  • A soma maior que 12 causa quebra de linha automática
  • Combine breakpoints para responsividade: .col-12 .col-md-6 .col-lg-4
  • Use diferentes larguras para criar layouts assimétricos

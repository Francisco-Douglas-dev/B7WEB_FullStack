│   └── style.css (customizações)

📄 HTML Básico

<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" 
        content="width=device-width, 
                 initial-scale=1.0">
  <title>Meu Site</title>
  <link href="bootstrap.min.css" 
        rel="stylesheet">
</head>
<body>
  <!-- Conteúdo -->
  <script src="bootstrap.bundle.min.js">
  </script>
</body>
</html>

Tipos de Containers

.container (largura fixa responsiva)
Este é um container com largura fixa que se adapta ao tamanho da tela. Possui margens laterais automáticas para centralização.
.container-fluid (largura 100%)
Este container ocupa 100% da largura da tela em qualquer tamanho de dispositivo.
.container-md (breakpoint específico)
Este container é fluid até atingir o breakpoint 'md', depois fica com largura fixa.

💡 Dicas Importantes:

  • Use .container para conteúdo centralizado com margens
  • Use .container-fluid para layouts de tela cheia
  • Sempre inclua a meta tag viewport para responsividade
  • Carregue o CSS no <head> e o JS antes de </body>
  • Use CDN para desenvolvimento rápido ou baixe para produção
⚠️ Ordem de Carregamento
  1. Bootstrap CSS (no head)
  2. Seu CSS customizado (depois do Bootstrap)
  3. Bootstrap JS (antes de fechar body)
  4. Seu JavaScript (por último)

📝 Classes de Botões

  • .btn .btn-{cor} - Botão colorido
  • .btn-outline-{cor} - Botão com contorno
  • .btn-lg / .btn-sm - Tamanhos
  • .btn-group - Grupo horizontal
  • .btn-group-vertical - Grupo vertical
  • .btn-toolbar - Barra de ferramentas
  • .d-grid - Botão em largura total
  • .active / disabled - Estados

📝 Atributos do Tooltip

  • data-bs-toggle="tooltip" - Ativa o tooltip
  • data-bs-title="..." - Texto do tooltip
  • data-bs-placement="top|right|bottom|left" - Posição
  • data-bs-html="true" - Permite HTML no tooltip
  • data-bs-trigger="hover|click|focus" - Como ativar
⚠️ Inicialização JavaScript Necessária

Tooltips precisam ser inicializados no JavaScript. Veja o código abaixo:

// Inicializar todos os tooltips
const tooltipTriggerList = document.querySelectorAll('[data-bs-toggle="tooltip"]');
const tooltipList = [...tooltipTriggerList].map(tooltipTriggerEl => new bootstrap.Tooltip(tooltipTriggerEl));

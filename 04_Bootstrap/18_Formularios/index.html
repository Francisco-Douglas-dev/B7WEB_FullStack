📝 Classes de Formulários

  • .form-control - Input de texto, email, senha, etc

  • .form-check - Checkbox/radio
  • .form-switch - Switch estilo iOS
  • .form-range - Input range/slider
  • .form-floating - Labels flutuantes
  • .form-text - Texto de ajuda
  • .is-valid / .is-invalid - Estados de validação

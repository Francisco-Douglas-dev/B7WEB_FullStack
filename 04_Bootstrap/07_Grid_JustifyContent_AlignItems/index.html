💡 Dicas

  • justify-content: controla espaçamento horizontal entre colunas
  • align-items: controla alinhamento vertical de todas as colunas
  • align-self: controla alinhamento vertical de uma coluna específica
  • Combine ambos para centralizar perfeitamente elementos
  • Use breakpoints: .justify-content-md-center

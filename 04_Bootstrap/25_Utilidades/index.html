📝 Categorias de Utilities

  • Cores: text-{color}, bg-{color}
  • Display: d-none, d-block, d-inline, d-flex
  • Flexbox: justify-content-*, align-items-*
  • Espaçamento: m-*, p-*, mt-*, mb-*, mx-*, my-*
  • Bordas: border, border-{side}, border-{color}
  • Border Radius: rounded, rounded-circle, rounded-pill
  • Texto: text-center, text-uppercase, fw-bold, fs-*
  • Dimensões: w-*, h-* (25, 50, 75, 100)
  • Sombras: shadow, shadow-sm, shadow-lg
  • Posição: position-*, top-*, bottom-*, start-*, end-*
💡 Dica Pro

Utilities são super poderosas! Use-as para criar layouts rapidamente sem precisar escrever CSS customizado. Combine múltiplas utilities para resultados impressionantes!

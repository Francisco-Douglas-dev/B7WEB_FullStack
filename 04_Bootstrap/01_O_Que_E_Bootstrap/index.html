Introdução ao Bootstrap

📚 O que é?

Bootstrap é um framework front-end gratuito e open-source para desenvolvimento web. Ele fornece uma coleção de ferramentas CSS e JavaScript para criar sites e aplicações responsivas rapidamente.

🎯 Para que serve?

  • Criar sites responsivos que funcionam em qualquer dispositivo
  • Acelerar o desenvolvimento com componentes prontos
  • Manter consistência visual em todo o projeto
  • Utilizar um sistema de grid flexível

⚙️ Como usar?

Via CDN (mais rápido):

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>

Via NPM:

npm install bootstrap

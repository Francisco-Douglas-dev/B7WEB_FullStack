Console do Navegador

🖥️ O que é o Console?

O console do navegador é uma ferramenta de desenvolvimento que permite:

  • Visualizar erros JavaScript
  • Executar código JavaScript em tempo real
  • Inspecionar elementos e seus estilos
  • Debugar aplicações

Como Abrir o Console

Windows/Linux
F12 ou Ctrl + Shift + I
Mac
Cmd + Option + I
Botão Direito

Clicar com o botão direito → Inspecionar

console.log()

Exibe mensagens no console

console.log("Olá, Console!");
console.error()

Exibe mensagens de erro

console.error("Erro!");
console.warn()

Exibe avisos

console.warn("Atenção!");
console.table()

Exibe dados em formato de tabela

console.table([{nome: "João"}]);
Inspecionando Elementos Bootstrap

Clique com o botão direito neste botão e selecione "Inspecionar":

O que observar no console:

  • Classes Bootstrap aplicadas (.btn, .btn-lg)
  • Estilos CSS computados
  • Propriedades do elemento
  • Event listeners (se houver)

✏️ Exercício Prático

Abra o console agora (F12) e:

  1. Clique nos botões acima para ver as mensagens
  2. Digite no console: document.body.style.backgroundColor = "lightblue"
  3. Inspecione o botão colorido e veja suas propriedades
  4. No console, digite: bootstrap para ver o objeto Bootstrap

Comandos Úteis no Console

Comando Descrição
document.querySelector('.classe') Seleciona um elemento
document.querySelectorAll('.classe') Seleciona todos os elementos
$0 Referencia o último elemento inspecionado
clear() Limpa o console
console.dir(elemento) Mostra propriedades do elemento

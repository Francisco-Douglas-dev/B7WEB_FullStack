Sistema de Grid do Bootstrap

📐 O que é o Grid System?

O Grid System do Bootstrap é baseado em 12 colunas e utiliza Flexbox. Ele permite criar layouts responsivos facilmente dividindo a tela em linhas (rows) e colunas (columns).

Estrutura Básica

<div class="container">
    <div class="row">
        <div class="col">Coluna 1</div>
        <div class="col">Coluna 2</div>
        <div class="col">Coluna 3</div>
    </div>
</div>

Container: Envolve todo o conteúdo
Row: Cria uma linha horizontal
Col: Define as colunas dentro da linha

💡 Dicas Importantes

  • Sempre envolva as .row dentro de .container ou .container-fluid
  • Colunas devem estar sempre dentro de .row
  • A soma das colunas deve ser 12 (ou menos, deixando espaço vazio)
  • Use classes responsivas para adaptar o layout a diferentes telas
  • O Grid usa Flexbox, então é muito flexível!

📝 Classes do Dropdown

  • .dropdown - Container principal
  • .dropdown-toggle - Botão que abre o menu
  • .dropdown-menu - O menu em si
  • .dropdown-item - Cada item do menu
  • .dropdown-header - Cabeçalho de seção
  • .dropdown-divider - Linha divisória
  • .dropup / .dropend / .dropstart - Direção do menu
  • .dropdown-menu-end - Alinha menu à direita
  • .dropdown-menu-dark - Tema escuro
  • data-bs-toggle="dropdown" - Inicializa o dropdown

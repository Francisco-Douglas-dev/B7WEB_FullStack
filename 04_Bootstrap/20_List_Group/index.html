📝 Classes do List Group

  • .list-group - Container principal
  • .list-group-item - Item da lista
  • .list-group-item-action - Para links e botões
  • .list-group-flush - Remove bordas
  • .list-group-horizontal - Lista horizontal
  • .list-group-numbered - Lista numerada
  • .list-group-item-{color} - Cores contextuais
  • .active - Estado ativo
  • .disabled - Estado desativado

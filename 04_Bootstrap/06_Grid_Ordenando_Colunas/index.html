Classes de Ordenação Disponíveis

Classe Descrição
.order-0 Primeira posição
.order-1 até .order-5 Posições específicas
.order-first Move para o início (order: -1)
.order-last Move para o final (order: 6)
.order-sm-*, .order-md-*, etc. Ordenação responsiva por breakpoint

💡 Dicas Importantes

  • Use .order- para reordenar colunas sem mudar o HTML
  • .order-first e .order-last são atalhos úteis
  • Combine com breakpoints para layouts responsivos inteligentes
  • Valores vão de 0 a 5, mas também aceita first e last
  • Útil para priorizar conteúdo em mobile sem duplicar HTML

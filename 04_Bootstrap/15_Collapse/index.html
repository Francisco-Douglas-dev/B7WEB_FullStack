📝 Atributos do Collapse

  • data-bs-toggle="collapse" - Ativa o comportamento de collapse
  • data-bs-target="#id" - ID do elemento a expandir/recolher
  • .collapse - Classe básica do elemento colapsável
  • .collapse.show - Inicia expandido
  • .collapse-horizontal - Collapse horizontal
  • .multi-collapse - Para controlar múltiplos elementos

📝 Classes Utilizadas

  • .d-flex - Cria um container flexbox
  • .flex-shrink-0 - Impede que o item encolha (para imagens)
  • .flex-grow-1 - Permite que o item cresça ocupando espaço disponível
  • .ms-3 / .me-3 - Margin start/end (espaçamento)
  • .align-items-start/center/end - Alinhamento vertical

📝 Classes da Navbar

  • .navbar - Container principal
  • .navbar-expand-{breakpoint} - Quando expandir (lg, md, sm)
  • .navbar-light / .navbar-dark - Tema claro ou escuro
  • .navbar-brand - Logo/marca
  • .navbar-toggler - Botão de toggle mobile
  • .navbar-nav - Container dos links
  • .nav-item - Item de navegação
  • .nav-link - Link de navegação
  • .fixed-top / .fixed-bottom / .sticky-top - Posicionamento
  • Combine com bg-{color} para cores
💡 Dica: Responsividade

Use .navbar-expand-lg para colapsar em telas menores que large. O botão .navbar-toggler aparece automaticamente em mobile!

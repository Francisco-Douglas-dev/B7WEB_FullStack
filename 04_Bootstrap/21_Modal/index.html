📝 Classes do Modal

  • .modal - Container principal
  • .modal-dialog - Container do conteúdo
  • .modal-content - Conteúdo do modal
  • .modal-header / .modal-body / .modal-footer - Seções
  • .modal-sm / .modal-lg / .modal-xl - Tamanhos
  • .modal-fullscreen - Tela cheia
  • .modal-dialog-centered - Centralizado
  • .modal-dialog-scrollable - Scroll interno
  • data-bs-toggle="modal" - Abre o modal
  • data-bs-dismiss="modal" - Fecha o modal

📝 Opções do Carousel

  • .carousel - Container principal
  • .carousel-inner - Wrapper dos slides
  • .carousel-item - Cada slide
  • .carousel-indicators - Indicadores de posição
  • .carousel-caption - Legenda sobre a imagem
  • .carousel-fade - Transição com fade
  • .carousel-dark - Tema escuro dos controles
  • data-bs-ride="carousel" - Iniciar automaticamente
  • data-bs-interval="3000" - Intervalo em ms
⚙️ Controle via JavaScript
// Obter instância do carousel
var myCarousel = document.querySelector('#carouselBasic')
var carousel = new bootstrap.Carousel(myCarousel)

// Métodos disponíveis
carousel.next()      // Próximo slide
carousel.prev()      // Slide anterior
carousel.pause()     // Pausar
carousel.cycle()     // Continuar
carousel.to(2)       // Ir para slide específico

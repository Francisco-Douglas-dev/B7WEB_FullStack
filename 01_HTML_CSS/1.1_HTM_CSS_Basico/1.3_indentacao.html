O que é indentação?

Indentação é o recuo das linhas de código para representar hierarquia e melhorar a legibilidade.

Exemplo sem e com indentação

Sem indentação (difícil de ler):

<html><body><div><p>Olá</p></div></body></html>

Com indentação (mais legível):

<html>
  <body>
    <div>
      <p>Olá</p>
    </div>
  </body>
</html>

Dica: use 2 ou 4 espaços por nível e mantenha padrão consistente.

Webpack + Sass

Integrando Sass/SCSS no Webpack

Por que usar Sass com Webpack?

Sass (Syntactically Awesome Stylesheets) é um pré-processador CSS que adiciona recursos poderosos como variáveis, aninhamento, mixins e funções. Com Webpack, o processo de compilação é automatizado e integrado ao build.

Instalação

# Instalar sass-loader e sass
npm install --save-dev sass-loader sass

# Também precisamos dos loaders de CSS
npm install --save-dev style-loader css-loader

Configuração do webpack.config.js

module.exports = {
    module: {
        rules: [
            {
                test: /\.s[ac]ss$/i,
                use: [
                    // Cria tags style no DOM
                    'style-loader',
                    // Transforma CSS em CommonJS
                    'css-loader',
                    // Compila Sass para CSS
                    'sass-loader'
                ]
            }
        ]
    }
};

// Os loaders são executados de baixo para cima:
// sass-loader → css-loader → style-loader

Exemplo de Uso

1. Criar arquivo styles.scss

// styles.scss
$primary: #1c78c0;
$secondary: #8b5cf6;

.container {
    padding: 20px;
    background: $primary;

    .title {
        color: white;
        font-size: 2rem;
        
        &:hover {
            color: $secondary;
        }
    }
}

2. Importar no JavaScript

// index.js
import './styles.scss';

console.log('Sass está funcionando!');

Recursos do Sass no Webpack

Variáveis

Use $nome para criar variáveis reutilizáveis

Aninhamento

Estruture CSS de forma hierárquica e organizada

Mixins

Crie blocos de código reutilizáveis com @mixin

Partials e Import

Organize estilos em múltiplos arquivos com @import

Funções

Use funções nativas como lighten(), darken(), etc.

Configurações Avançadas

{
    test: /\.s[ac]ss$/i,
    use: [
        'style-loader',
        'css-loader',
        {
            loader: 'sass-loader',
            options: {
                // Opções do sass-loader
                sassOptions: {
                    indentWidth: 4,
                    includePaths: ['src/styles']
                }
            }
        }
    ]
}

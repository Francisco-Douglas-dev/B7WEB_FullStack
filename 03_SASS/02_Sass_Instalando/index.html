Instalando o Sass

Guia completo de instalação

1. Instalação via NPM

npm install -g sass

Instala o Sass globalmente no seu sistema

2. Verificar instalação

sass --version

Verifica se o Sass foi instalado corretamente

3. Compilar arquivo SCSS

sass input.scss output.css

Compila um arquivo SCSS para CSS

Agora você está pronto para usar Sass! 🎉

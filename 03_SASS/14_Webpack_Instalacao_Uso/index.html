Webpack: Instalação e Uso

Primeiros passos com o empacotador de módulos

O que é Webpack?

Webpack é um empacotador de módulos (module bundler) para aplicações JavaScript modernas. Ele processa e combina múltiplos arquivos em bundles otimizados.

Instalação

# Instalar webpack globalmente
npm install -g webpack webpack-cli

# Ou localmente no projeto
npm install --save-dev webpack webpack-cli

Uso Básico

Comando Simples:

npx webpack

Este comando irá procurar por um arquivo de entrada padrão e gerar o bundle.

Com Parâmetros:

npx webpack --mode development

Define o modo de compilação (development ou production).

Vantagens do Webpack

  • Gerenciamento de dependências automatizado
  • Otimização de código para produção
  • Suporte a diversos tipos de arquivos
  • Hot Module Replacement (HMR)
  • Code splitting e lazy loading

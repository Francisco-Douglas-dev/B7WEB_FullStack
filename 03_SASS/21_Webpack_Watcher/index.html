Webpack Watcher

Modo watch para recompilação automática

O que é o Watch Mode?

O modo watch do Webpack monitora mudanças nos arquivos do projeto e recompila automaticamente quando detecta alterações. Isso elimina a necessidade de executar o build manualmente a cada modificação.

Ativando o Watch Mode

Método 1: Via Linha de Comando

# Executar webpack em modo watch
npx webpack --watch

# Ou usando yarn
yarn webpack --watch

Método 2: package.json Script

{
    "scripts": {
        "build": "webpack",
        "watch": "webpack --watch",
        "dev": "webpack --watch --mode development"
    }
}

# Executar
npm run watch

Método 3: webpack.config.js

module.exports = {
    mode: 'development',
    watch: true,
    watchOptions: {
        ignored: /node_modules/,
        aggregateTimeout: 300,
        poll: 1000
    }
};

Opções de Configuração

ignored

Define pastas/arquivos que não devem ser monitorados (ex: node_modules)

ignored: /node_modules/

aggregateTimeout

Tempo de espera (ms) após uma mudança antes de recompilar

aggregateTimeout: 300

poll

Verifica mudanças em intervalos (útil em sistemas que não suportam eventos de arquivo)

poll: 1000

Webpack Dev Server (Alternativa Recomendada)

Para desenvolvimento, o webpack-dev-server é mais poderoso que o watch simples, oferecendo live reload e HMR.

# Instalar webpack-dev-server
npm install --save-dev webpack-dev-server

# package.json
{
    "scripts": {
        "start": "webpack serve --open",
        "dev": "webpack serve --mode development"
    }
}

# webpack.config.js
module.exports = {
    devServer: {
        static: './dist',
        port: 3000,
        hot: true,
        open: true
    }
};

Watch vs Dev Server

Watch Mode
Recompila arquivos automaticamente
Dev Server
Recompila + Live Reload + HMR + Servidor HTTP

Dicas de Uso

  • Use watch para builds simples e testes rápidos
  • Prefira dev server para desenvolvimento ativo
  • Sempre ignore node_modules para melhor performance
  • Configure aggregateTimeout para evitar recompilações excessivas
  • Use Ctrl+C para parar o processo watch

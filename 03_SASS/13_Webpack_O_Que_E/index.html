📦 O que é Webpack?

Module Bundler para aplicações JavaScript modernas

Definição

Webpack é um empacotador de módulos (module bundler) para JavaScript.

Ele pega múltiplos arquivos JS, CSS, imagens e outros assets, e os transforma em um ou mais bundles otimizados.

Principais Características

📦

Bundling

Empacota múltiplos arquivos em bundles menores

🔄

Code Splitting

Divide código em chunks carregados sob demanda

🎨

Loaders

Transforma arquivos (Sass, TypeScript, etc)

🔌

Plugins

Estende funcionalidades do Webpack

Por que usar Webpack?

  • ✅ Gerenciamento de dependências
  • ✅ Otimização automática
  • ✅ Hot Module Replacement (HMR)
  • ✅ Suporte a diversos formatos de arquivo
  • ✅ Tree shaking (remove código não usado)

🚀 Webpack torna o desenvolvimento web mais eficiente!

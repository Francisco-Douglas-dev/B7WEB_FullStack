Webpack: Carregando CSS

Como processar e carregar arquivos CSS com Webpack

CSS Loaders no Webpack

Para trabalhar com arquivos CSS no Webpack, precisamos de loaders específicos que processam e injetam os estilos na aplicação.

Instalação dos Loaders

# Instalar style-loader e css-loader
npm install --save-dev style-loader css-loader

# Ou com yarn
yarn add -D style-loader css-loader

css-loader: Interpreta @import e url() como import/require() e resolve-os.

style-loader: Injeta o CSS no DOM através de tags <style>.

Configuração no webpack.config.js

module.exports = {
    module: {
        rules: [
            {
                test: /\.css$/i,
                use: ['style-loader', 'css-loader']
            }
        ]
    }
};

// IMPORTANTE: A ordem dos loaders importa!
// Eles são executados de trás para frente:
// 1. css-loader processa o CSS
// 2. style-loader injeta no DOM

Como Usar

1. Criar arquivo CSS

Crie seu arquivo styles.css normalmente.

2. Importar no JavaScript

// No seu arquivo index.js
import './styles.css';

// O Webpack irá processar e injetar o CSS

3. Build e Resultado

Ao executar o build, o CSS será empacotado junto com o JavaScript e injetado dinamicamente na página.

Opções Avançadas

// Exemplo com CSS Modules
{
    test: /\.css$/,
    use: [
        'style-loader',
        {
            loader: 'css-loader',
            options: {
                modules: true,
                importLoaders: 1
            }
        }
    ]
}
